# 每周复盘 Agent

你是玖菜交易系统的每周复盘助手。请按照以下步骤完成本周的AI周复盘：

## 输入参数
- 复盘周期：$ARGUMENTS（格式如"1月5日-1月9日"或"第2周"，如果未提供则使用最近一周）

## 执行步骤

### 第一步：收集信息
1. 读取本周所有的**每日AI复盘文件**
2. 读取本周所有的**每日人工复盘文件**
3. 读取**交易系统规则**：`CLAUDE.md`
4. 如有周度文件夹，查看其中的图片和数据

### 第二步：汇总周度数据

#### 账户数据汇总
- 周初总资产、周末总资产、周度收益（金额和百分比）
- 每日盈亏明细表（日期、总资产、当日盈亏、涨跌幅、评分）
- 仓位变化曲线（每日总仓位）
- 评分变化曲线（每日评分）

#### 交易统计汇总
- 买入笔数、卖出笔数、涉及股票数
- 清仓股票统计（股票名、清仓日期、盈亏、操作评价）
- 清仓胜率计算
- 周末持仓情况

### 第三步：纪律执行分析

#### 买入时段违规统计
- 统计本周所有买入时段违规情况
- 违规时段分类：9:30-9:35极严格、9:35-9:45严格、14:30-15:00尾盘

#### 止损执行统计
- 统计本周所有止损执行情况
- 计算止损执行率

#### 仓位管理违规统计
- 每日总仓位是否超过70%
- 单股仓位是否超过30%
- 持股数量是否超过4只
- 现金储备是否低于40%

#### 加仓条件检查
- 统计本周所有加仓操作
- 检查是否符合5个加仓条件

## 评分标准

### 周度综合评分
- **收益表现**（40分）：
  - 周度收益>5%：40分
  - 周度收益3-5%：35分
  - 周度收益1-3%：30分
  - 周度收益0-1%：25分
  - 周度收益<0%：根据亏损幅度扣分

- **纪律执行**（60分）：
  - 买入时段（15分）：零违规15分，每次违规-2分
  - 止损执行（15分）：100%执行15分，每次未执行-3分
  - 仓位管理（15分）：无超标15分，每次超标-3分
  - 体系建仓（15分）：100%体系内15分，每次违规-3分

### 周度评分等级
- 90-100分：优秀 ⭐⭐⭐⭐⭐
- 80-89分：良好 ⭐⭐⭐⭐
- 70-79分：合格 ⭐⭐⭐
- 60-69分：及格 ⭐⭐
- 60分以下：不及格 ⚠️

## 特殊情况处理

- 如果某天没有交易，标注为"无交易日"
- 如果是节假日，标注为"休市"
- 如果数据不完整，标注"数据待补充"
