# 玖菜交易系统 MVP1.0

> 一个专为A股个人投资者设计的T+1交易系统，强调风险控制和纪律化交易

## 项目简介

这是一个完整的A股交易系统，适用于总资金约115,000元的个人投资者。系统采用严格的风险控制和纪律化交易策略，通过系统化的方法提升交易胜率和盈亏比。

**核心理念：** 控制风险，让盈利奔跑

## 核心特点

### 🛡️ 严格的风险控制
- **单笔最大亏损：** 575元（总资金的0.5%）
- **止损线：** -5%（或技术支撑位）
- **仓位上限：** 单股≤20%，总仓位≤60%
- **现金储备：** ≥40%

### 📈 清晰的盈利目标
- **主要目标：** +15%（盈亏比3:1）
- **普通股票三档止盈：** +8% / +15% / +25%
- **妖股动态减仓：** +30%起步，每涨15%减仓10%

### ⏰ 适配T+1规则
- **最佳买入时段：** 9:45-10:30
- **午盘风险管理：** 11:25前评估（应对13:00-13:30无法操作）
- **禁止买入时段：** 集合竞价、开盘混乱期、尾盘

### 🎯 双策略体系
1. **强势突破策略：** 捕捉放量突破的强势股
2. **回调买入策略：** 在关键支撑位买入优质回调股

### 系统模块

本交易系统包含以下模块：

1. **模块1：选股系统** - 建立自选股池，每日盘后筛选
2. **模块2：买入系统** - 买入时机和执行流程
3. **模块3：仓位管理** - 首次建仓、加仓规则
4. **模块4：止损系统** - 技术止损 + 百分比止损
5. **模块5：止盈系统** - 普通股三档止盈 + 妖股动态减仓
6. **模块6：每日操作流程** - 盘前、盘中、盘后完整流程
7. **模块7-9：检查清单** - 买入前、持仓管理、快速参考
8. **模块10-11：交易记录** - 日志模板、复盘模板

## 核心纪律

### ✅ 必须执行
- 只买自选股池的票
- 首次必须10%仓位
- 买入即设止损止盈
- 触发止损立即执行
- 触发止盈分批执行
- 11:25前评估午盘风险

### ❌ 绝对禁止
- 追涨停
- 买炸板股
- 亏损加仓
- 炸板后补仓
- 单股超20%
- 尾盘买入（14:30-15:00）
- 首次仓位>10%
- 当日大涨时追高加仓（涨幅>3%）⭐

## 系统优势

1. **简单易执行：** 规则清晰，不需要复杂判断
2. **风险可控：** 单笔最大亏损仅575元
3. **收益合理：** 主要目标+15%，盈亏比3:1
4. **适配T+1：** 特别考虑了A股T+1规则和午盘风险
5. **双轨策略：** 覆盖强势突破和回调买入两种场景
6. **心态优化：** 妖股策略解决"卖飞"心理问题，止盈后心态管理
7. **实战验证：** 基于真实交易经验持续优化，从理论到实战的成熟系统

## 版本历史

### v1.5 - 2025-12-23 ⭐ 最新
**实战优化版本**
- 🎯 加仓条件优化：新增"当日涨幅<3%"要求，避免追高
- 📝 建仓后加仓策略：等回调或横盘企稳再加仓
- 🔧 炸板回封策略：区分一次炸板（0-10分钟回封可观察）和二次炸板（立即止盈）
- 💎 止盈后心态管理：删除自选、隐藏持仓、不追回
- 🚫 盘中信息管理：不看论坛、不看群聊、信息越少操作越准
