# 玖菜交易系统 MVP1.0

> 一个专为A股个人投资者设计的T+1交易系统，强调风险控制和纪律化交易

## 项目简介

这是一个完整的A股交易系统，基于 **10万元总资金** 设计。系统采用严格的风险控制和纪律化交易策略，通过系统化的方法提升交易胜率和盈亏比。

**核心理念：** 控制风险，让盈利奔跑

**交易风格：** 短线趋势动量交易
- 🎯 持仓周期：3-15天
- 📈 右侧交易：跟随趋势，不预测拐点
- ⚡ 热点题材：捕捉板块轮动机会
- 🛡️ 严格风控：小亏大赚，盈亏比4:1~5:1
- 📋 纪律执行：量化清单，机械操作

## 核心特点

### 🛡️ 严格的风险控制
- **单笔最大亏损：** 500元（总资金的0.5%）
- **止损线：** -5%（或技术支撑位）
- **仓位上限：** 单股≤20%，总仓位≤60%
- **现金储备：** ≥40%

### 📈 灵活的盈利目标
- **主要目标：** 20~25%（盈亏比4:1~5:1）⭐
- **灵活三档止盈：**
  - 第一档：8~12%（看盘面，卖30%~40%）
  - 第二档：20~25%（看盘面，卖30%~40%）⭐
  - 第三档：趋势终结（剩余全部清仓）
- **妖股动态减仓：** +30%起步，每涨15%减仓10%

### ⏰ 适配T+1规则
- **最佳买入时段：** 9:45-10:30
- **午盘风险管理：** 11:25前评估（应对13:00-13:30无法操作）
- **禁止买入时段：** 集合竞价、开盘混乱期、尾盘

### 🎯 双策略体系
1. **强势突破策略：** 捕捉放量突破的强势股
2. **回调买入策略：** 在关键支撑位买入优质回调股

### 系统模块

本交易系统包含以下模块：

1. **交易系统核心要点** - 核心参数、五大支柱、关键数字、纪律清单
2. **模块1：选股系统** - 建立自选股池，每日盘后筛选
3. **模块2：买入系统** - 买入时机和执行流程
4. **模块3：仓位管理** - 首次建仓、加仓规则
5. **模块4：止损系统** - 技术止损 + 百分比止损
6. **模块5：止盈系统** - 灵活三档止盈 + 妖股动态减仓
7. **模块6：快速检查清单** - 每日操作、买入前检查、持仓管理、快速参考
8. **模块7：复盘模板** - 交易日志模板、周度/月度复盘

## 核心纪律

### ✅ 必须执行
- 只买自选股池的票
- 首次必须10%仓位
- 买入即设止损止盈
- 触发止损立即执行
- 触发止盈分批执行
- 11:25前评估午盘风险

### ❌ 绝对禁止
- 追涨停
- 买炸板股
- 亏损加仓
- 炸板后补仓
- 单股超20%
- 尾盘买入（14:30-15:00）
- 首次仓位>10%
- 当日大涨时追高加仓（涨幅>3%）⭐

## 系统优势

1. **简单易执行：** 规则清晰，不需要复杂判断
2. **风险可控：** 单笔最大亏损仅500元
3. **收益合理：** 主要目标20~25%，盈亏比4:1~5:1
4. **适配T+1：** 特别考虑了A股T+1规则和午盘风险
5. **灵活止盈：** 根据盘面强弱灵活调整，不设死板固定目标
6. **双轨策略：** 覆盖强势突破和回调买入两种场景
7. **心态优化：** 妖股策略解决"卖飞"心理问题，止盈后心态管理
8. **实战验证：** 基于真实交易经验持续优化，从理论到实战的成熟系统
